AtForms

A component to show form fields status , errors and a set of validators.

Note: At this time AtForm only works with model driven forms (Reactive forms)

Setup

Import the [AtFormsModule] in your NgModule:

Usage

Use at-form-field as a wrapper for your input. Provide a "label" to display as a "fieldLabel" on Error messages or before the input fields if you want horizontal forms.

How to use AtValidators

Setup

Import the [AtFormsModule] in your NgModule:

my.module.ts

Usage

Import AtValidators in your component and pass one or a combination of AtValidators to the form control. Each AtValidator support a custom message to display in case of errors.
